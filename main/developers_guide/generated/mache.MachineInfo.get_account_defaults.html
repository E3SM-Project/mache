mache.MachineInfo.get_account_defaults

MachineInfo.get_account_defaults()[source]

Get default account, partition and quality of service (QOS) for this machine.

Returns:

  • account (str) – The E3SM account on the machine

  • partition (str) – The default partition on the machine, or None if no partition should be specified

  • constraint (str) – The default constraint on the machine, or None if no constraint should be specified

  • qos (str) – The default quality of service on the machine, or None if no QOS should be specified
